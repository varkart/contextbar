# Product

## Register

product

## Users

Solo developers on macOS who run 2+ AI coding tools simultaneously (Claude Code, Cursor, Gemini CLI, Copilot, Windsurf, ChatGPT). Power users who live in the terminal and their editor. They have the menu bar populated. They know exactly what they want.

## Product Purpose

Single-click visibility into all installed AI tooling on the machine. Answers "what's loaded, what's configured, and what MCPs are active" without opening four different config files. Lives in the menu bar, stays out of the way.

## Brand Personality

Sharp, minimal, opinionated. Confident like a developer tool that knows what it's doing. No decorative flourish — every pixel serves information.

## Anti-references

- Generic SaaS dark dashboards: purple gradients, glassy hero cards, metric callouts
- IDE config panels: tree views, heavy checkbox lists, VS Code settings
- Electron-heavy apps: bloated dark UIs (Slack, Discord) that feel sluggish and foreign on macOS

## Design Principles

1. **Information first.** Layout exists to present data, never to impress. No chrome for chrome's sake.
2. **Native or nothing.** Feels like it belongs on macOS. System fonts, system-adjacent colors, respects the OS aesthetic.
3. **Dense but breathable.** Packs data into 380×520px without feeling cramped. Rhythm through consistent micro-spacing, not padding inflation.
4. **Opinionated defaults.** No toggles for things that should just work. The UI makes choices; users don't configure the UI.
5. **Quiet confidence.** No color noise. Accent color earns its place. Status is communicated in one glance.

## Accessibility & Inclusion

WCAG AA. Reduced motion supported. All interactive elements keyboard navigable.
